## 3. 对局页

```text
+------------------------------------------------------+
| 顶部状态条: 房间 / 轮次 / 当前玩家 / 阶段            |
+-------------------------------+----------------------+
| 我的计分纸主视图              | 右侧信息区           |
|                               | 公共骰池             |
| 这里优先保留原版计分纸语言    | 银盘                 |
| 使用 SVG + HTML 可点击热区    | 玩家摘要             |
|                               | 操作面板             |
+-------------------------------+----------------------+
```

## 4. 设计约束

- 自己的计分纸是主视觉，不做实体桌面俯视模拟
- 公共骰池和银盘放到视觉中心附近
- 其他玩家信息默认摘要化
- 奖励链反馈要明显，但不要挡住主视图
- 桌面端优先，平板端次之，手机端后补
